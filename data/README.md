# Data Description

This folder contains input data for ENSO prediction.

## Data Sources

The datasets used in this study are publicly available:

- Niño-3.4 SST anomaly  
  https://psl.noaa.gov/data/timeseries/month/data/nino34.long.anom.data

- Southern Oscillation Index (SOI)  
  https://www.cpc.ncep.noaa.gov/data/indices/soi

- Warm Water Volume (WWV)  
  https://www.pmel.noaa.gov/tao/wwv/data/wwv.dat

## Instructions

Download the datasets from the links above and place them in this folder before running the scripts.

Ensure consistent formatting as required by the preprocessing scripts.
